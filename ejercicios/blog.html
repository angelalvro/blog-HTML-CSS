Layout Index

EL layout general está configurado de tal forma que disponemos principalmente de tres elementos.

  • Header: Ocupa todo el ancho de la página con un color de texto negro sobre un amarillo verdoso
  • Main: Dispone de display flex a través de la clase contenido-principal. Está compuesto de tres artículos, uno para cada ejercicio
  • Footer: Ocupa todo el ancho de la página con un color de texto negro sobre un amarillo verdoso

Layout de las plantillas ejercicios

EL layout de las tres plantillas de los ejercicios sigue la misma filosofía que el layout general. Podemos destacar, que en este caso el display de los artículos es el la dirección de columna, de manera que las imágenes se sitúan debajo de los textos explicativos.

Estructura

Dentro del blog se ha usado tanto selectores como clases, primándose las clases ya que se aprovecha mejor el código CSS desarrollado. De esta forma se obliga a realizar el marcado semántico muy estructurado.

Por otro lado, el blog tiene una estructura semántica, validándose el código HTML. Solo nos encontramos con algún Warnig debido a la falta de encabezados en algunos de los artículos. En nuestro caso el encabezado esta embebido en el sub artículo.
